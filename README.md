# Yi OpenClaw Skills 🧰

个人 OpenClaw Skills 集合，可直接安装使用。

## Skills

| Skill | 说明 |
|-------|------|
| **jargon-translator** 🎭 | 互联网黑话双向转换器 |
| **communication-coach** 🥊 | 苏格拉底式挑刺 + 降维表达 |
| **pm-requirement-flow** 📋 | PM 工作流：需求澄清 → 派发 → 验收 |

## 快速安装

```bash
# 安装单个 skill
openclaw skills add https://raw.githubusercontent.com/yiguoguo/yi-openclaw-skills/main/pm-requirement-flow/SKILL.md
```

## 前提条件

### 1. OpenClaw

参考：https://openclaw.ai

### 2. agency-agents（推荐）

PM 工作流参考了 [agency-agents](https://github.com/msitarzewski/agency-agents) 的角色定义。

```bash
cd <your-openclaw-workspace>

```
